用 Rand7() 实现 Rand10()

已有方法 rand7 可生成 1 到 7 范围内的均匀随机整数，试写一个方法 rand10 生成 1 到 10 范围内的均匀随机整数。

思路：

class Solution extends SolBase {
    public int rand10() {
        int first, second;
        while((first = rand7()) > 6);
        while((second = rand7()) > 5);
        return first % 2 == 0 ? second : 5 + second;
    }
}
